Fundamental Bahasa Pemrograman Dart

Pengenalan Dart

Dart adalah bahasa pemrograman yang dikembangkan oleh Google, dirancang khusus untuk pengembangan aplikasi client-side. Dart mendukung paradigma Object-Oriented Programming (OOP) dan memiliki syntax yang familiar bagi developer yang sudah menguasai Java, JavaScript, atau C#.

1. Variables dan Data Types

Variables.dart 
// Deklarasi variable dengan type inference
var nama = 'John Doe';           // String
var umur = 25;                   // int
var tinggi = 175.5;              // double
var isStudent = true;            // bool

// Deklarasi variable dengan explicit type
String alamat = 'Jakarta';
int jumlahSKS = 144;
double ipk = 3.75;
bool lulus = false;

// Konstanta
final String universitas = 'Universitas Indonesia';  // Runtime constant
const double pi = 3.14159;                           // Compile-time constant

// Nullable variables (null safety)
String? nomorTelepon;  // Bisa null
int? nilaiUjian;       // Bisa null

// Late initialization
late String tokenAPI;  // Akan diinisialisasi nanti

void main() {
  print('Nama: $nama');
  print('Umur: $umur tahun');
  print('Tinggi: $tinggi cm');
  print('Status Mahasiswa: $isStudent');
}

2. Functions

Functions.dart 
// Function dasar
String sapaPengguna(String nama) {
  return 'Halo, $nama! Selamat datang di Flutter Course.';
}

// Function dengan optional parameters
double hitungIPK(double totalNilai, int jumlahMatkul, [double bonus = 0.0]) {
  return (totalNilai + bonus) / jumlahMatkul;
}

// Function dengan named parameters
String buatProfilMahasiswa({
  required String nama,
  required String nim,
  int semester = 1,
  String? jurusan,
}) {
  return 'Nama: $nama, NIM: $nim, Semester: $semester, Jurusan: ${jurusan ?? "Belum ditentukan"}';
}

// Arrow function (untuk function sederhana)
double luasLingkaran(double radius) => 3.14159 * radius * radius;

// Function sebagai first-class object
Function tambah = (int a, int b) => a + b;

void main() {
  // Penggunaan functions
  print(sapaPengguna('Ahmad'));
  print('IPK: ${hitungIPK(320.0, 8)}');
  
  var profil = buatProfilMahasiswa(
    nama: 'Sari Dewi',
    nim: '2021001234',
    semester: 5,
    jurusan: 'Teknik Informatika'
  );
  print(profil);
  
  print('Luas lingkaran: ${luasLingkaran(7.0)}');
  print('Hasil penjumlahan: ${tambah(5, 3)}');
}

3. Control Flow

ControlFlow.dart 
void main() {
  // If-else statement
  double ipk = 3.75;
  String predikat;
  
  if (ipk >= 3.75) {
    predikat = 'Summa Cum Laude';
  } else if (ipk >= 3.50) {
    predikat = 'Magna Cum Laude';
  } else if (ipk >= 3.25) {
    predikat = 'Cum Laude';
  } else {
    predikat = 'Memuaskan';
  }
  print('Predikat kelulusan: $predikat');

  // Switch-case statement
  int semester = 5;
  String status;
  
  switch (semester) {
    case 1:
    case 2:
      status = 'Mahasiswa Baru';
      break;
    case 3:
    case 4:
      status = 'Mahasiswa Tingkat Menengah';
      break;
    case 5:
    case 6:
      status = 'Mahasiswa Tingkat Atas';
      break;
    case 7:
    case 8:
      status = 'Mahasiswa Tingkat Akhir';
      break;
    default:
      status = 'Status Tidak Dikenal';
  }
  print('Status: $status');

  // For loop
  print('\nDaftar Semester:');
  for (int i = 1; i <= 8; i++) {
    print('Semester $i');
  }

  // Enhanced for loop (for-in)
  List mataKuliah = ['Algoritma', 'Database', 'Web Programming', 'Mobile Programming'];
  print('\nMata Kuliah:');
  for (String matkul in mataKuliah) {
    print('- $matkul');
  }

  // While loop
  int jumlahSKS = 0;
  int targetSKS = 144;
  int semesterSekarang = 1;
  
  while (jumlahSKS < targetSKS) {
    jumlahSKS += 20; // Asumsi 20 SKS per semester
    print('Semester $semesterSekarang: Total SKS = $jumlahSKS');
    semesterSekarang++;
  }
  
  print('Lulus setelah ${semesterSekarang - 1} semester!');
}

Arsitektur Flutter dan Rendering Engine

Arsitektur Flutter

Flutter menggunakan arsitektur berlapis yang terdiri dari tiga layer utama:

Layer Arsitektur Flutter

Framework Layer
(Dart)
Widgets, Animation, Gestures
Engine Layer
(C++)
Skia Graphics, Dart Runtime
Platform Layer
(Platform-specific)
iOS, Android, Web, Desktop

Rendering Engine

  • Skia Graphics Engine - Engine grafis yang sama digunakan Chrome dan Android
  • Dart VM - Menjalankan kode Dart dengan Just-In-Time (JIT) compilation untuk development
  • AOT Compilation - Ahead-Of-Time compilation untuk production builds
  • Widget Tree - Representasi hierarkis dari UI components

Keunggulan Arsitektur Flutter

  1. Performa Native - Tidak menggunakan bridge seperti React Native
  2. Konsistensi UI - Tampilan sama di semua platform
  3. Hot Reload - Perubahan code langsung terlihat tanpa restart
  4. Customizable - Kontrol penuh terhadap setiap pixel

Instalasi Flutter SDK

System Requirements

Minimum: Intel i5/Ryzen 5, 8GB RAM, 256GB SSD, Windows 10/macOS Catalina/Ubuntu 18.04

Recommended: Intel i5 10th gen+/Ryzen 5, 16GB RAM, 512GB SSD

Langkah-langkah Instalasi

1. Download Flutter SDK
Download Commands 
# Windows - PowerShell
Invoke-WebRequest -Uri https://storage.googleapis.com/flutter_infra_release/releases/stable/windows/flutter_windows_3.24.5-stable.zip -OutFile flutter_windows.zip

# macOS - Terminal
curl -O https://storage.googleapis.com/flutter_infra_release/releases/stable/macos/flutter_macos_3.24.5-stable.zip

# Linux - Terminal
wget https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_3.24.5-stable.tar.xz

echo 'export PATH="$PATH:/path/to/flutter/bin"' >> ~/.bashrc

Implementasi "Hello Indonesia" App

Complete Code: main.dart

lib/main.dart 
import 'package:flutter/material.dart';

// Entry point aplikasi Flutter
void main() {
  runApp(HelloIndonesiaApp());
}

// Root widget aplikasi
class HelloIndonesiaApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Hello Indonesia',           // Judul aplikasi
      theme: ThemeData(
        primarySwatch: Colors.red,        // Warna merah (bendera Indonesia)
        visualDensity: VisualDensity.adaptivePlatformDensity,
      ),
      home: HomePage(),                   // Halaman utama
      debugShowCheckedModeBanner: false, // Hilangkan debug banner
    );
  }
}

// Widget halaman utama
class HomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text(
          'Hello Indonesia 🇮🇩',
          style: TextStyle(
            fontWeight: FontWeight.bold,
            color: Colors.white,
          ),
        ),
        backgroundColor: Colors.red,
        elevation: 4.0,
      ),
      body: Container(
        // Background gradient dengan warna bendera Indonesia
        decoration: BoxDecoration(
          gradient: LinearGradient(
            begin: Alignment.topCenter,
            end: Alignment.bottomCenter,
            colors: [
              Colors.red.shade100,
              Colors.white,
              Colors.red.shade100,
            ],
          ),
        ),
        child: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // Icon Indonesia
              Icon(
                Icons.location_on,
                size: 80.0,
                color: Colors.red,
              ),
              SizedBox(height: 20.0),
              
              // Judul utama
              Text(
                'Selamat Datang di',
                style: TextStyle(
                  fontSize: 24.0,
                  fontWeight: FontWeight.w300,
                  color: Colors.grey[700],
                ),
              ),
              
              // Judul Indonesia
              Text(
                'INDONESIA',
                style: TextStyle(
                  fontSize: 36.0,
                  fontWeight: FontWeight.bold,
                  color: Colors.red,
                  letterSpacing: 2.0,
                ),
              ),
              
              SizedBox(height: 30.0),
              
              // Card informasi
              Container(
                margin: EdgeInsets.symmetric(horizontal: 40.0),
                padding: EdgeInsets.all(20.0),
                decoration: BoxDecoration(
                  color: Colors.white,
                  borderRadius: BorderRadius.circular(15.0),
                  boxShadow: [
                    BoxShadow(
                      color: Colors.grey.withOpacity(0.3),
                      spreadRadius: 2,
                      blurRadius: 10,
                      offset: Offset(0, 5),
                    ),
                  ],
                ),
                child: Column(
                  children: [
                    _buildInfoRow('Negara:', 'Republik Indonesia'),
                    _buildInfoRow('Ibu Kota:', 'Jakarta'),
                    _buildInfoRow('Jumlah Pulau:', '17.508 pulau'),
                    _buildInfoRow('Populasi:', '270+ juta jiwa'),
                    _buildInfoRow('Bahasa:', 'Bahasa Indonesia'),
                  ],
                ),
              ),
              
              SizedBox(height: 30.0),
              
              // Tombol aksi
              ElevatedButton(
                onPressed: () {
                  // Action ketika tombol ditekan
                  _showWelcomeDialog(context);
                },
                child: Text(
                  'Jelajahi Indonesia',
                  style: TextStyle(
                    fontSize: 18.0,
                    fontWeight: FontWeight.bold,
                  ),
                ),
                style: ElevatedButton.styleFrom(
                  backgroundColor: Colors.red,
                  foregroundColor: Colors.white,
                  padding: EdgeInsets.symmetric(horizontal: 30.0, vertical: 15.0),
                  shape: RoundedRectangleBorder(
                    borderRadius: BorderRadius.circular(25.0),
                  ),
                  elevation: 5.0,
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
  
  // Widget helper untuk membuat baris informasi
  Widget _buildInfoRow(String label, String value) {
    return Padding(
      padding: EdgeInsets.symmetric(vertical: 5.0),
      child: Row(
        mainAxisAlignment: MainAxisAlignment.spaceBetween,
        children: [
          Text(
            label,
            style: TextStyle(
              fontWeight: FontWeight.bold,
              color: Colors.grey[700],
            ),
          ),
          Text(
            value,
            style: TextStyle(
              color: Colors.red,
              fontWeight: FontWeight.w500,
            ),
          ),
        ],
      ),
    );
  }
  
  // Function untuk menampilkan dialog
  void _showWelcomeDialog(BuildContext context) {
    showDialog(
      context: context,
      builder: (BuildContext context) {
        return AlertDialog(
          title: Text('Selamat Datang! 🎉'),
          content: Text(
            'Terima kasih telah menggunakan aplikasi Hello Indonesia. '
            'Ini adalah aplikasi Flutter pertama Anda!',
          ),
          actions: [
            TextButton(
              child: Text('OK'),
              onPressed: () {
                Navigator.of(context).pop(); // Tutup dialog
              },
            ),
          ],
        );
      },
    );
  }
}

Daftar Istilah dan Singkatan

AOT (Ahead-Of-Time) Compilation
Proses kompilasi kode sebelum runtime, menghasilkan performa lebih baik untuk production builds.
API (Application Programming Interface)
Antarmuka yang memungkinkan komunikasi antara berbagai komponen software.
AVD (Android Virtual Device)
Emulator Android yang berjalan di komputer untuk testing aplikasi mobile.
Cross-Platform Development
Pendekatan pengembangan aplikasi yang dapat berjalan di multiple platform dari satu codebase.
Dart
Bahasa pemrograman yang dikembangkan Google, digunakan untuk membuat aplikasi Flutter.
Flutter SDK
Software Development Kit yang berisi tools, libraries, dan documentation untuk Flutter development.
Hot Reload
Fitur Flutter yang memungkinkan developer melihat perubahan code secara real-time tanpa restart aplikasi.
JIT (Just-In-Time) Compilation
Proses kompilasi kode saat runtime, digunakan Flutter untuk development mode dengan hot reload.
Material Design
Design system yang dikembangkan Google untuk menciptakan user interface yang konsisten dan intuitif.
Native Development
Pengembangan aplikasi menggunakan bahasa dan tools khusus untuk platform tertentu (Swift untuk iOS, Java/Kotlin untuk Android).
Scaffold
Widget Flutter yang menyediakan struktur dasar halaman (AppBar, Body, FloatingActionButton, dll).
Skia
Graphics engine open-source yang digunakan Flutter untuk rendering, juga digunakan Chrome dan Android.
Widget
Building block utama Flutter UI, setiap elemen visual dalam Flutter adalah widget.
Widget Tree
Struktur hierarkis widget-widget dalam aplikasi Flutter, menentukan layout dan appearance UI.
